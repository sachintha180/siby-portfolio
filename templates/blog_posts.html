27-05-2023: Welcome to my portfolio!

Hi, this is my first blog post. Nothing much, but its honest work. In this post, I wanted to mention a few courses that I've found to be instrumental in my progress. These courses are all available free-of-charge, and teach you a variety of digital skills. Here's the list:

  • CS50x by Harvard

    : To help you learn everything at an introductory computer science level. From binary & basic Python skills, to data structures in C++ and web development. 10/10, would recommend!

  • CS50ai by Havard

    : For those interested in artificial intelligence, a subtle but comprehensive course on fundementals of machine learning and walks you through content such as searching algorithms, classifications & regression and neural networks.

  • Nand to Tetris by UofJerusalem

    : An exciting course on computer engineering and hardware, where the goal of the course is to teach you how to go from Nand-gates to a fully functional computer that can run tetris. Teaches its content via an amazing hardware simulator, so no actual circuit building necessary. Amazing stuff!

  • Full Stack Open by UoHelsinki

    : An entire course on full-stack-development using the Javascript, MongoDB, ReactJS / React Native and many other tools. Extremely comprehensive IMO.

  • Trainee Full Stack Developer Programme by UofMorotuwa

    : A very comprehensive course on full-stack-development, teaching you skills all the way from basic Python to web development tools. The learning curve is very shallow, and would recommend for anyone wishing to get into the IT industry.

In addition to these course, Reddit & StackOverflow have been my go-to sites for reviews and help on course content. These are the courses that I'm currently taking, and therefore have a lot more to understand and review. Nevertheless, it has been an amazing journey up until now, and hope I'll be able to pull through until the end. Stay tuned for my next update, best of luck to you!

29-05-2023: Portfolio update #2

I've now completed the skills.html page with a full Bootstrap-based grid for showcasing my technical skills, and an unordered list with sliders for showcasing my additional courses + their completion.

An interesting feature that was implemented was a custom sorting function - sortSkills() - for sorting my technical skills in descending order of proficiency.

The "badge effect" for the proficiency of each skill was achieved using box-shadow & border-radius properties, with a translucent border-color.

All content for the skills, courses and the responsibilities list (in index.html) were added dynamically to the HTML page using jQuery. This data was initialized in JavaScript objects and arrays, both 1 and 2 dimensional.

Thank you for sticking around. Hoping to publish my portfolio into a .com domain soon!
